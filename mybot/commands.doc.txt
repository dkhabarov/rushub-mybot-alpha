ИМЯ:
	rmmyaccount

СИНТАКСИС:
	+rmmyaccount пароль [ -h ]

ОПИСАНИЕ:
	rmmyaccount - Удаляет аккаунт пользователя, который вызвал команду.
	В данной команде надо обязательно указать пароль от своего аккаунта.

ОПЦИИ:
	-h	 Показать эту справку

ПРИМЕР ИСПОЛЬЗОВАНИЯ:
	+rmmyaccount mysuperpassword - Удалить аккаунт. Вместо mysuperpassword надо указать свой пароль.

ИМЯ:
	+regme

СИНТАКСИС:
	+regme [ пароль ] [ -h ]

ОПИСАНИЕ:
	regme - Регестрирует пользователя, который вызвал команду c паролем [ пароль ]. 
	Пароль надо указывать через пробел.

Правила регистрации ника:
	Минимальная длина пароля: 4
	Максимальная длина пароля: 30
	

ОПЦИИ:
	-h	 Показать эту справку

ПРИМЕР ИСПОЛЬЗОВАНИЯ:
	+regme mysuperpassword
	Вместо mysuperpassword укажите свой пароль.

ИМЯ:
	bindip

СИНТАКСИС:
	+bindip nick ipv4 [ -h ]

ОПИСАНИЕ:
	bindip - Устанавливает привязку IP-адреса ipv4 для пользователя с ником nick.
	В данное время поддерживаются только IPv4 адреса. Юзер nick должен быть зарегистрированным.
	Если пользователь с ником nick онлайн, и IP-адрес не соответствует установленному, он будет отключен.

ОПЦИИ:
	-h	 Показать эту справку

ПРИМЕР ИСПОЛЬЗОВАНИЯ:
	+bindip testuser 192.168.0.2 - Установить привязку ip-адреса 192.168.0.2 для юзера testuser.

ИМЯ:
	passwd

СИНТАКСИС:
	+passwd старый пароль новый пароль [ -h ]

ОПИСАНИЕ:
	passwd - Меняет пароль пользователя, который вызвал команду.
	Для смены пароля надо указать сначала старый пароль, а потом через пробел новый.

ОПЦИИ:
	-h	 Показать эту справку

ПРИМЕР ИСПОЛЬЗОВАНИЯ:
	+passwd oldpass newpass - Сменить старый пароль oldpass на новый newpass. Вместо oldpass и newpass надо указать свои пароли.
